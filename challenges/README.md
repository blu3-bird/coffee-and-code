# 🎯 Git Practice Challenges

Test your Git skills with these hands-on challenges!

## Beginner Challenges

### Challenge 1: First Steps
- [ ] Create a new branch called `feature/my-first-change`
- [ ] Add a new HTML file with your name
- [ ] Commit with a meaningful message
- [ ] Push and create a PR

### Challenge 2: Branch Navigation
- [ ] Create 3 different branches
- [ ] Make a small change in each
- [ ] Switch between them and observe the changes
- [ ] Merge them into main

### Challenge 3: Commit History
- [ ] Make 5 commits with clear messages
- [ ] Use `git log` to view your history
- [ ] Practice `git show` to see commit details

## Intermediate Challenges

### Challenge 4: Conflict Resolution
- [ ] Edit the same line in a file on two different branches
- [ ] Try to merge and resolve the conflict
- [ ] Complete the merge successfully

### Challenge 5: Undo Changes
- [ ] Make a change and commit it
- [ ] Use `git revert` to undo the commit
- [ ] Understand the difference between revert and reset

### Challenge 6: Stashing
- [ ] Make changes without committing
- [ ] Use `git stash` to save them
- [ ] Switch branches and come back
- [ ] Apply your stashed changes

## Advanced Challenges

### Challenge 7: Interactive Rebase
- [ ] Create 3 small commits
- [ ] Use `git rebase -i` to squash them
- [ ] Rewrite commit messages

### Challenge 8: Cherry-Pick
- [ ] Create a commit on one branch
- [ ] Cherry-pick it to another branch
- [ ] Verify the commit was applied

### Challenge 9: Full Workflow
- [ ] Fork this repo (if not already done)
- [ ] Create a feature branch
- [ ] Make multiple commits
- [ ] Rebase onto latest main
- [ ] Create a PR
- [ ] Respond to code review comments
- [ ] Get your PR merged

---

## Checklist for All Challenges

Before considering a challenge complete:
- [ ] My code works correctly
- [ ] Commit messages are clear and descriptive
- [ ] I understand what each Git command does
- [ ] I've reviewed my changes before committing

**Remember:** The goal is learning, not perfection. Ask questions when stuck!
