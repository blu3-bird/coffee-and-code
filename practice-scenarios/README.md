# Git & GitHub Practice Scenarios

This folder contains structured practice scenarios for learning Git and GitHub workflows.

## Available Scenarios

### 1. Basic Git Operations
- Creating your first commit
- Making changes and committing
- Viewing commit history

### 2. Branching Practice
- Creating feature branches
- Switching between branches
- Merging branches

### 3. Collaboration Workflows
- Forking repositories
- Creating pull requests
- Code review process
- Resolving merge conflicts

### 4. Advanced Git
- Rebasing
- Stashing changes
- Resetting and reverting
- Cherry-picking commits

## How to Use

1. Choose a scenario that matches your learning goal
2. Follow the instructions in the scenario file
3. Practice the Git commands
4. Submit your work for review

---

**Tip:** Don't be afraid to make mistakes! That's how we learn.
