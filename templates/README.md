# Templates

This folder contains starter templates for your practice projects.

## Available Templates

### 1. Project Template (`project-template.html`)
A basic HTML5 template with:
- Semantic HTML structure (header, main, footer)
- Navigation menu
- Multiple sections
- Responsive design ready

### 2. Styles Template (`styles.css`)
A CSS starter file with:
- Reset and base styles
- Header and navigation styling
- Main content area styling
- Footer styling
- Responsive media queries

## How to Use

1. Copy the template files to your project folder
2. Customize the content
3. Add your own styles or modify existing ones
4. Practice Git workflows as you build!

## Tips

- Start with the template and make it your own
- Experiment with different layouts
- Try adding new features
- Don't forget to commit your changes frequently!
